# esirko's OpenSCAD models

This repo contains simple [OpenSCAD](https://openscad.org/) models that I've made to solve simple problems. I'm learning this fresh so I'm sure most of these models are janky in both their solution to the problem and code!

## actually useful or otherwise interesting

[actually useful or otherwise interesting](./actually%20useful%20or%20otherwise%20interesting/README.md)

Models that I made that are useful to me, may be useful to others (with or without tweaking), or otherwise interesting to talk about. I typically have pictures, demos, and commentaries of these.

## failures but good effort

[failures but good effort](./failures%20but%20good%20effort/README.md)

My horror file of attempts to do things that ultimately didn't work and I gave up. These may or may not have commentary... but usually not.

## funny one-offs

[funny one-offs](./funny%20one-offs/README.md)

Sometimes the solution is a bit ridiculous, so I put these here for browsing. Typically have pictures and commentary, but I don't expect them to be useful to anyone.

## obscure one-offs

[obscure one-offs](./obscure%20one-offs/README.md)

dumping ground for obscure one-offs that I don't feel like writing commentary for, but weren't really failures either so I can't put them in the failures directory.
