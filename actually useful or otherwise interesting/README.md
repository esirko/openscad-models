## 2025-03-01 keyboard mouse blocker

I use a Kinesis split keyboard (Freestyle Edge) with a lift kit that raises the halves of the keyboard by a small angle for ergonomics. I also use a Logitech MX Master 3 mouse. When moving the mouse, I found that I frequently hit the left mouse button against the keyboard, inadvertantly clicking the mouse whereever it happens to be on screen. This is super annoying. So I decided to build a guard to prevent the mouse button from being hit. This took a few iterations to get the dimensions right.

With the print in place and attached to the keyboard wrist pad from the underside, the part of the mouse _underneath_ the button will collide with the print first. This prevents the left mouse button from being inadvertantly clicked.
